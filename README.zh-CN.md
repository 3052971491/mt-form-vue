## 简介

拖拽快速开发vue页面

## 功能

- **特性一**：拖拽式可视化表单设计
- **特性二**：支持运行时动态加载表单
- **特性三**：支持表单复杂交互控制
- **特性四**：支持全局CSS样式, 全局函数, 全局变量

## 安装使用

- 获取项目代码

```bash
git clone https://github.com/3052971491/mt-form-vue.git
```

- 下载依赖

```bash
cd mt-form-vue

yarn install

```

- 打包

```bash
yarn lib
```

- 运行

```bash
yarn dev
```

- 使用

```bash
npm install mt-form-vue
or
yarn add mt-form-vue
```
